# with-game-master
Akashic Engineを利用して作った、ニコ生上で動かすためのマルチプレイ対応ゲームのサンプル

# how to use
Akashic Engineの開発環境を整えたあと以下を実行します

```
npm install
npm run build
akashic serve
```
